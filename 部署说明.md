# NetBird 快速部署说明（新版优先）

本文件给出 5 分钟部署入口，默认走官方最新脚本。

## 一、推荐执行

```bash
export NETBIRD_DOMAIN=netbird.example.com
curl -fsSL https://github.com/netbirdio/netbird/releases/latest/download/getting-started.sh | bash
```

安装完成后，浏览器访问：

```text
https://netbird.example.com
```

如页面卡住，先检查：

- 域名是否解析到正确服务器
- 80/443 是否可公网访问
- `curl`/`docker` 是否已安装

## 二、常见问题（快速排障）

- 无法登录管理端：检查 DNS、端口、证书日志。
- 客户端显示 NAT 问题：检查 3478/udp 是否开放。
- 需要企业 IdP：直接运行官方 `getting-started-with-zitadel.sh`，不要依赖本地脚本。
